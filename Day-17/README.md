# Day 17 - Quiz Project

A simple quiz game built with Python using Object-Oriented Programming principles.

## Description

This project implements a trivia quiz game that asks users multiple-choice questions and tracks their score. It demonstrates the use of classes, objects, and data management in Python.

## Features

- Multiple-choice trivia questions
- Score tracking
- Question bank management
- User interaction through console

## Project Structure

- `quiz_project.py` - Main program file
- `question_model.py` - Question class definition
- `quiz_brain.py` - Main program file,Quiz logic and game flow
- `data.py` - Question data storage

## How to Run

```bash
python quick_project.py
```

## Concepts Learned

- Object-Oriented Programming (OOP)
- Class creation and instantiation
- Working with attributes and methods
- Data modeling
- Program flow control

## Requirements

- Python 3.x

## Usage

Run the program and answer the questions by typing your answers. The program will track your score and display the final results at the end.
